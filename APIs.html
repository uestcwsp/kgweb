API:Zhishi.me provides an online API which has 4 query modes as illustrated below. Users can query the knowledge graph by sending HTTP request executed using the GET method.The results will be given in a Json string.
Query all knowledge of a given entity
  • URL: zhishi.me/api/entity/[label]
  • Example: zhishi.me/api/entity/中华人民共和国
Query all knowledge from a specific online encyclopedia of a given entity
  • URL: zhishi.me/api/entity/[label]?baike=[encyclopedia]
  • Example: zhishi.me/api/entity/中华人民共和国?baike=baidubaike
Query all knowledge of an entity's given property
  • URL: zhishi.me/api/entity/[label]?property=[property]
  • Example: zhishi.me/api/entity/中华人民共和国?property=infobox
There are 10 properties available to inquiry, they are listed as below:


property explanation
abstracts Abstractions of the entity from Baidubaike, Hudongbaike and Chinese Wikipedia
infobox Infobox properties in an entity's corresponding online encyclopedia pages
category Categories of the entity's corresponding articles in Baidubaike, Hudongbaike and Chinese Wikipedia
Query all knowledge from a specific online encyclopedia of entity's given property
  • URL: zhishi.me/api/entity/[label]?property=[property]&baike=[encyclopedia]
  • Example: zhishi.me/api/entity/中华人民共和国?property=infobox&baike=baidubaike
This query mode just combines the above three modes.
